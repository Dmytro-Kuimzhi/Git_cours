Dir - Показывает список обьектов в папке
Status - Показывает статус обьектов
add Файл - Добавляет в список отслеживаемых
rm --cached Файл - Удаляет из списка отслеживаемых
commit -m "Коментарий" - Фиксирует версию
branch - Показывает в какой ветке мы находимся
branch Название - Создает новую ветку
branch -D Название - Удалить ветку
checkout Название - Переключится на другую ветку 
checkout -b Название - Создать новую ветку и переключится на нее
